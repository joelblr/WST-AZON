# WST-AZON
Web Scraper Tool For Amazon Reviews Dataset Generation

##### Git
```bash
git clone git@github.com:joelblr/WST-AZON.git
cd WST-AZON\
```
##### Nodejs
```bash
mkdir amazon_reviews
cd amazon_reviews
npm init -y
npm install puppeteer
```
##### Python
```bash
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
python main.py
```
